# Flow Demo: Media Upload Module

> Демонстрация полного цикла разработки с AI — от бизнес-требований до работающего кода.

## Структура демо

```
docs/examples/media-upload/
├── FLOW-README.md              ← этот файл
├── 01-REQUIREMENTS.md          ← Шаг 1: бизнес-требования (пишет человек, 3 строки)
├── 02-SPEC.md                  ← Шаг 2: AI превращает требования в техзадание
├── 03-CODE.md                  ← Шаг 3: AI генерирует код + тесты
└── 04-VERIFICATION.md          ← Шаг 4: верификация (тесты, curl, review)
```

## Как читать

Это **живой шаблон**. Не абстракция — пример того, как выглядит каждый шаг
для реальной фичи. Если заменить `media-upload` на любую другую фичу,
структура останется той же.
